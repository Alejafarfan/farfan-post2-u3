## Descripción del proyecto

Este proyecto consiste en el desarrollo de un **dashboard web responsivo** utilizando **CSS3**, aplicando los modelos modernos de layout: **CSS Grid** para la estructura principal de la página y **Flexbox** para la organización de los componentes internos.

El dashboard incluye:

* Sidebar de navegación con enlaces.
* Barra superior (topbar) con título y acciones.
* Tarjetas de estadísticas responsivas.
* Tabla de proyectos recientes.
* Panel de actividad reciente.

El objetivo del proyecto es demostrar el uso correcto de **Grid y Flexbox sin utilizar frameworks CSS externos**, creando una interfaz funcional y adaptable a diferentes tamaños de pantalla.

## Instrucciones para ejecutar el proyecto

1. Clonar o descargar el repositorio.
2. Abrir la carpeta del proyecto en **Visual Studio Code**.
3. Instalar o activar la extensión **Live Server**.
4. Abrir el archivo **index.html**.
5. Hacer clic derecho y seleccionar **Open with Live Server**.

El proyecto se abrirá automáticamente en el navegador.

## Características del diseño

* Uso de **CSS Grid** para el layout principal.
* Uso de **Flexbox** para organizar sidebar, topbar y componentes internos.
* Diseño **responsivo** con repeat(auto-fill, minmax()).
* Uso de **Custom Properties (variables CSS)** para colores y tamaños.
* Implementación de **BEM** para la nomenclatura de clases.
